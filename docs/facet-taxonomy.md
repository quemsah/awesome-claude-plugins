# Facet Taxonomy

Category and keyword filters are generated from checked-in repository names and descriptions until a static plugin manifest index is available.

## Categories

- Agent workflows: agent, swarm, workflow, orchestration, automation.
- MCP and integrations: MCP, server, integration, connector, API.
- Skills and prompts: skill, prompt, command, template.
- Developer tooling: code, review, testing, lint, git, CLI.
- Knowledge and memory: memory, RAG, context, knowledge, docs.
- Security and operations: security, guardrail, compliance, monitoring, observability.
- Design and content: design, UI, slides, image, video.

## Keywords

Keyword filters use common terms from the same text source: Claude, MCP, agent, skill, prompt, workflow, memory, security, design, data, testing, and review.

This taxonomy is deterministic and intentionally conservative. It should be replaced or supplemented by plugin manifest categories and keywords once those manifests are statically indexed.
